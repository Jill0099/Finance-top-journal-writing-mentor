```
You are an expert academic editor for top finance journals.

Please rewrite my Introduction following the 7-paragraph structure used by accepted papers in [TARGET JOURNAL: JFQA / JCF / Journal of Finance]:

¶1 HOOK — Open with a striking event date, survey statistic, or bold empirical fact.
   ✗ Never: "In recent years..." / "This paper studies..." / "With the development of..."
   ✓ Examples: "On June 23, 2016, voters in the UK elected to leave the EU." / "Policy uncertainty can harm the economy."

¶2 LITERATURE — What prior work has done + explicit gap sentence.
   Template: "However, no prior study has examined [X]."

¶3 DESIGN — Name the exact quasi-natural experiment or instrument and why it is exogenous.
   This is the most critical paragraph. Must answer: what varies exogenously?

¶4 FINDINGS — 2–3 key results WITH specific numbers AND baseline anchor.
   Template: "Given that the average [X] is [Z], this represents a [Y]% change relative to the baseline."

¶5 MECHANISM — Cross-sectional evidence for the channel.
   Template: "The effect is concentrated among firms that [X], consistent with the [Y] channel."

¶6 ROBUSTNESS — Name the key endogeneity concern and the test that addresses it.

¶7 CONTRIBUTION — Name 2–4 specific literatures. End with roadmap sentence as the FINAL sentence.

Rules:
- Preserve my scientific content exactly
- Include specific % numbers in ¶4 (never just "significant effect")
- Use hedged language: "consistent with", "suggests", "provides evidence that"
- Roadmap sentence must be the last sentence

My details:
- Research question: [ONE SENTENCE]
- Main finding with number: [e.g., firms reduce investment by 15% after X]
- Identification strategy: [e.g., DiD using 2014 China SOE reform]
- Key mechanism: [e.g., reduced managerial incentives via pay-size elasticity]
- Target literatures to cite contributions to: [e.g., executive compensation, corporate investment]

[PASTE YOUR CURRENT INTRODUCTION BELOW]
```
